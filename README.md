# Kryptovox 🐺

> *Hidden voices. Made audible.*

Academic analysis of bipedal cryptid encounter testimony — cross-referenced against global mythology and historical record spanning thousands of years.

## What this is

The world's first structured, large-scale analysis of anomalous bipedal creature witness testimony. We apply forensic psychology credibility tools, NLP pattern detection, and historical folklore comparison to answer one question:

**Why does this description persist — consistently, independently, across cultures with no apparent contact — for thousands of years?**

## Phenomena studied
- **Dogman** — bipedal canine, primary corpus: Dogman Encounters (Vic Cundiff)
- **Bigfoot / Sasquatch** — bipedal primate, corpus: BFRO, Sasquatch Chronicles, Bigfoot Society

## The pipeline
```
YouTube → yt-dlp (audio) → Whisper (transcript) → Claude (extraction) → Supabase (database) → Analysis
```
